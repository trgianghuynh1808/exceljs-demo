# ExcelJS Demo

## Description

Project using `ExcelJS` package to build Import/Export Excel file feature on web browser.

## Run Source With Development Mode

1. yarn install
2. yarn dev

## Run Source With Production Mode

1. yarn install
2. yarn build
3. yarn start
